Roadmap
Here's a small preview of the things we're currently focusing on, as well as some of our plans for future projects. Did you spotted a bug using bonfire or you want to propose a new feature ?
Open an issue Join our Matrix chat
In progress
  • Federation
    Implement basic federation for social activities and moderation. We will focus on testing federation between different bonfire instances and between bonfire and mastodon.
    Read more
  • Release Candidate (0.5.0) issues list
    Use the campground instance to test and catch all the bugs we need to fix before calling our first Bonfire Release Candidate.
    Read more
  • Circle & Boundaries
    Ensure Circles & Boundaries flow works smooth and without bugs
    Read more
  • Improve Developer Experience
    Ease the onboarding experience for new developers by improving docs, writing guides and create good first issues.
Planned
  • Groups Alpha Implementation
    At its core, Bonfire is a set of configurations that define what extensions to use and how they play together.
  • Groups Federation
    Pick or create a theme that reflects your community identity, edit and localise terminology to increase inclusivity, decide which kind of information and behaviours
  • Roles
    Connect with other people and communities in a decentralized environment, including other fediverse platforms that support the ActivityPub protocol (like Mastodon, PeerTube, etc...)
  • Federated topics
    A customizable federated social network, to follow people, make posts, have discussion, etc...
    Read more
